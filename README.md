# 潍坊好吃好玩地图

一个面向山东潍坊的“好吃、好玩、可探索”网页地图。当前版本采用手绘动态地图表达：地点信息保持真实、可追溯，地图底图不直接复制第三方真实瓦片。

## 当前方向

- 只推荐值得放入地图的地点，不做拉踩或负面排名。
- 地点数据集中维护在 `places.js`，包含地址、电话、开放信息、标签、图片来源和公开来源链接。
- 地图视觉是自绘的纸感/卡通风格，带动态路线、脚印、路牌点位、缩放、拖拽和巡游切换。
- 开启“编辑”后可以拖动路牌位置，导出的是手绘布局坐标，不会改真实经纬度。

## 本地预览

```text
node server.js
```

浏览器打开：

```text
http://127.0.0.1:5173/
```

不要直接双击 `index.html` 作为最终验证方式；本项目包含本地静态服务，使用上面的地址更稳定。

## 数据与版权边界

- 地点事实信息应优先来自公开、可核验来源，例如官方页面、公开经营信息、地图平台公开 POI 信息、票务/景区公告等。
- 当前界面不使用高德地图瓦片作为视觉底图，因此不会把高德地图画面直接复制到产品里。
- 如果后续接入高德、Mapbox、MapTiler 等服务，需要遵守对应平台的开发者协议、配额、Logo/版权标注和 Key 安全配置。
- 可以参考“动态手绘地图”的交互思路，但不要使用《哈利波特》原作名称、图案、素材、字体或高度相似的视觉版式。
