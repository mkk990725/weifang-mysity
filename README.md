# 潍坊好吃好玩地图

这是一个关于山东潍坊的城市探索地图项目，计划整理潍坊本地值得吃、值得逛、值得玩的地点，做成一份方便查看和分享的城市指南。

## 项目目标

- 收集潍坊特色美食、餐馆、小吃街和咖啡茶饮地点
- 整理适合游玩、散步、拍照和亲子出行的城市目的地
- 用地图形式展示地点位置、分类、推荐理由和实用信息
- 后续支持按区域、类型、评分或关键词筛选

## 内容方向

### 好吃的

- 潍坊本地特色小吃
- 老字号餐馆
- 烧烤、火锅、鲁菜、家常菜
- 咖啡馆、甜品店、夜宵地点

### 好玩的

- 城市公园和广场
- 博物馆、文化街区和展馆
- 适合周末出行的景区
- 适合拍照、散步和亲子游的地点

## 真实地图接入准备

当前网页已支持高德地图 JS API，用于展示国内真实路网、楼块和 POI 底图。要继续增强为更完整的真实三维地图，需要先确定地图供应商和准备以下信息：

- Web 端地图 API Key，以及供应商要求的安全配置，例如域名白名单或安全密钥
- 正式部署域名，用于配置 key 的来源限制
- 需要的地图能力：标准路网、卫星影像、3D 建筑、3D 地形、POI 搜索、路线规划
- 坐标体系选择和转换方案：国内地图服务通常使用 GCJ-02，OpenStreetMap/MapLibre 常用 WGS84
- 调用量预估和商业授权预算，避免上线后超额或违反服务条款

可选方向：

- 高德地图 JS API：适合国内真实路网、POI、搜索、路线规划和 3D 地图展示
- Mapbox 或 MapTiler：适合国际化 WebGL 地图、3D 建筑、卫星影像和自定义视觉风格
- Cesium：适合更重的真实三维地球、地形、倾斜摄影或 3D Tiles 场景

## 本地配置

项目已支持高德地图 JS API。当前 `config.js` 中放的是 Web JS API Key，前端 Key 在浏览器里本来就是可见的；上线前请在高德开放平台配置域名白名单和安全密钥，降低被滥用风险。

如果需要换成自己的 Key，修改：

```js
window.WEIFANG_MAP_CONFIG = {
  amapKey: "YOUR_AMAP_WEB_JS_KEY",
  amapSecurityJsCode: ""
};
```

如果高德后台开启了安全密钥校验，把对应的安全密钥填到 `amapSecurityJsCode`。
