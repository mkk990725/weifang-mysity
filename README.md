# 潍坊好吃好玩翻页地图

这是一个把潍坊真实地点信息做成翻页书体验的网页项目。当前实现使用 `StPageFlip` / `page-flip` 这一类页面翻转引擎，把地图、点位和详情拆成封面、总览页、地点页和封底。

## 特点

- 真实地点信息保留地址、电话、开放时间、票务和公开来源。
- 视觉上采用手绘纸感、路网、路牌和立体建筑，不直接复制真实底图。
- 总览页的路牌可点击跳转到具体地点页。
- 支持按分类和关键字筛选，翻页内容会随筛选结果更新。
- 开启“编辑”后可以拖动总览页路牌，导出的是翻页书布局坐标。

## 本地运行

```text
node server.js
```

浏览器打开：

```text
http://127.0.0.1:5173/
```

## 技术边界

- 当前页面使用的是页面翻转引擎，不是传统地图瓦片。
- 仍然要遵守地点来源的真实性和版权边界。
- 如果后续想继续接入更强的地图交互，可以把总览页和地点页继续扩展成多章节书页。
